# Connectify - Modern Dating App

A production-ready, advanced dating application built with Flask and modern web technologies.

## Features

- **User Authentication**: Secure login, registration, and password reset
- **Profile Management**: Complete user profiles with verification
- **Matching System**: Advanced algorithm for matching users
- **Real-time Chat**: WhatsApp-style messaging with typing indicators and read receipts
- **Dashboard**: User analytics and settings
- **Notifications**: Real-time notification system
- **Monetization**: Subscription plans and payment integration
- **Safety Features**: Block, report, and privacy controls
- **Chatbot**: AI-powered assistant for user guidance
- **Responsive Design**: Works on all device sizes
- **Dark Mode**: Toggle between light and dark themes

## Technology Stack

- **Backend**: Flask with Blueprints for modularity
- **Frontend**: Jinja2 templates, HTML5, CSS3, JavaScript (ES6)
- **Database**: SQLAlchemy with SQLite (can be easily switched to PostgreSQL/MySQL)
- **Real-time Communication**: Flask-SocketIO
- **Authentication**: Flask-Login
- **Styling**: Custom CSS with CSS variables for theming

🌍 Travel Buddy – Connect and Explore Together

Travel Buddy is a web application designed to help solo travelers find compatible travel companions based on their destination, travel dates, and interests. The platform encourages safer, more enjoyable, and cost-effective travel by connecting people with similar travel plans.

✨ Features
🔐 Secure User Authentication
👤 User Profile Management
🤝 Travel Companion Matching
📍 Destination-Based Search
💬 Real-Time Chat & Messaging
📅 Travel Event Creation & Joining
🤖 AI Travel Assistant
🛡️ Admin Dashboard for User & Content Management
📱 Responsive Design for Desktop and Mobile

🎯 Project Objective

The main goal of Travel Buddy is to connect travelers with compatible companions while improving travel safety, reducing costs, and making trip planning easier through intelligent matching and AI-powered assistance.

🚀 Key Benefits
Find travel partners with similar interests
Plan trips more efficiently
Improve travel safety
Share travel expenses

# Travel Buddy

This repository contains the Travel Buddy Flask application and related utilities.

## Project structure

- `tbapp/` – main application source and utilities
- `tbapp/App/` – application package, routes, templates, and database helpers

## Local setup

1. Create and activate a virtual environment.
2. Install dependencies from the project's requirements file.
3. Run the Flask application from the app entrypoint.

## Notes

- The repository is configured to ignore local virtual environments and secrets.
- For production use, copy `.env.example` to `.env` and provide your environment-specific values.

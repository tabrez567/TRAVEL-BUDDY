# 🌍 Travel Buddy - Connect and Explore Together

Travel Buddy is a full-stack web application that helps solo travelers connect with compatible travel companions based on their destination, travel dates, and shared interests. The platform makes travel safer, more enjoyable, and cost-effective through intelligent matching, real-time communication, and AI-powered travel assistance.

# 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/Travel-Buddy.git
```

### Navigate to Project

```bash
cd Travel-Buddy
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

Windows

```bash
venv\Scripts\activate
```

Linux / Mac

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

Open your browser:

```
http://127.0.0.1:5000
```
